Завдання 3: Робота з методами масивів

1. Сортування масиву за роком видання

const sortedByYear = [...library].sort((a, b) => a.year - b.year); console.log("Відсортовано за роком видання:"); sortedByYear.forEach(book => console.log(book.bookInfo()));

2. Фільтрація непрочитаних книг

const unreadBooks = library.filter(book => !book.isRead); console.log("Непрочитані книги:"); unreadBooks.forEach(book => console.log(book.bookInfo()));

3. Пошук книги автора "J.R.R. Tolkien"

const tolkienBook = library.find(book => book.author === "J.R.R. Tolkien"); console.log("Книга автора J.R.R. Tolkien:"); if (tolkienBook) { console.log(tolkienBook.bookInfo()); } else { console.log("Книгу не знайдено"); }
